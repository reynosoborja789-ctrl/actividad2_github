# Guía básica

## ¿Qué es Git?
Git es un sistema de control de versiones que permite guardar cambios en archivos y llevar un historial de versiones de un proyecto.

## ¿Qué es GitHub?
GitHub es una plataforma en la nube que permite alojar repositorios Git y compartirlos con otras personas.

## Diferencia entre repositorio local y remoto
Un repositorio local está en mi ordenador y lo uso para trabajar directamente.
Un repositorio remoto está en Internet, por ejemplo en GitHub, y sirve para guardar copias, sincronizar cambios y compartir el proyecto.
